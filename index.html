Verb To Be
Para tornar o Verb To Be negativo,
basta adicionar a palavra Not.
am not, is not, are not.
Podemos usar estas contrações:
isn't / aren't
São geralmente usadas depois de substantivos. Ou estas:
I'm not..., You're not..., He's not..., She's not..., We're not..., They're not...
For example:
I___ happy today.

You___ sad now.

He___ a teacher. He's a mechanic.

She___ tired. She's energetic.

It___ sunny this morning. It's rainy.

We___ brothers. We're cousins.

They___ friendly. They're rude.
